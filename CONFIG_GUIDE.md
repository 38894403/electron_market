# 配置文件使用指南

## 配置文件位置

TYPE_2数据现在从系统级配置文件读取，配置文件位置：

**Windows系统：**
```
C:\Users\[用户名]\.market-app\type2_config.json
```

**macOS系统：**
```
/Users/[用户名]/.market-app/type2_config.json
```

**Linux系统：**
```
/home/[用户名]/.market-app/type2_config.json
```

## 配置文件格式

配置文件为JSON格式，示例内容：

```json
{
  "TYPE_2": [
    {
      "code": "225.lcm",
      "text": "碳酸锂"
    },
    {
      "code": "114.lhm", 
      "text": "生猪"
    },
    {
      "code": "122.XAU",
      "text": "金-美"
    }
  ]
}
```

## 使用方法

1. **首次运行**：应用会自动创建配置文件目录和默认配置文件
2. **修改配置**：直接编辑配置文件，保存后重启应用即可生效
3. **配置验证**：应用启动时会验证配置文件格式，如果格式错误会使用默认配置

## 注意事项

- 配置文件必须是有效的JSON格式
- 每个配置项必须包含 `code` 和 `text` 字段
- 修改配置文件后需要重启应用才能生效
- 如果配置文件不存在或格式错误，应用会使用默认配置
